# scraping_service


[Скрапинг сервис](https://job-pars.herokuapp.com) на Джанго для сбора вакансий. 

Сервис который собирает данные о новых вакансиях из нескольких источников.
Пользователь может посмотреть свежие вакансии в своем городе по интересующему направлению, а так же
подписаться на рассылку интересующих вакансий.
